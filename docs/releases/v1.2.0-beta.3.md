# App Monitor 1.2.0 beta 3

This release brings App Monitor's update manager much closer to a complete native update workflow while retaining the cleanup, appearance, and self-update improvements already merged into `main`.

## Highlights

- Finds updates from the Mac App Store, Homebrew, Sparkle, app metadata, Electron, GitHub, and manual sources.
- Adds working source filters, update-row selection, a details inspector, and in-app formatted release notes.
- Opens apps and triggers their built-in updater for guided manual updates without downloading unusable Sparkle delta payloads.
- Adds Homebrew cask adoption, one-click **Adopt & Update All**, and future updates through Homebrew.
- Requests administrator authorization once and stores it in the user's macOS Keychain through a signed askpass helper. The saved credential can be removed from Settings.
- Reconciles completed App Store and guided updates when App Monitor becomes active again.

## Reliability Fixes

- Clears stale update records after an app has actually been updated or adopted.
- Avoids Homebrew cask downgrades and repairs missing cask artifacts when possible.
- Uses compatible Homebrew formula flags and handles cask adoption conflicts more accurately.
- Excludes Xcode DerivedData copies from installed-app update discovery.
- Keeps full Sparkle enclosure URLs while filtering delta-only payloads from guided downloads.
